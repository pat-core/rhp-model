# Rotating Heat Pipe Model
Unverified implementation of

SONG, F.; EWING, D.; CHING, C. Y. 
_Fluid flow and heat transfer model for high-speed rotating heat pipes._ 
International Journal of Heat and Mass Transfer, 2003, 46(23), 4393-4401.
https://doi.org/10.1016/S0017-9310(03)00292-8

in MatLab and translated to Python.
